This website is designed to help Chinese language learners enhance their oral reading skills. The text-to-speech feature is not limited to Chinese but supports other languages as well. It utilizes your browser's capabilities, and hence the quality of the speech depends on the device and web browser you are using. When you open this website on the Microsoft Edge browser on a Windows PC, it has the most options of natural voices to choose from.

Text to Speech

This text-to-speech feature utilizes the built-in capabilities of web browsers for speech synthesis. For the best quality and most natural-sounding speech experience, open this web page using the Microsoft Edge browser on a Windows PC.

Note:

For iOS users, you can select either Tingting(zh-CN) or Meijia (zh-TW) for Chinese text-to-speech synthesis. For Android users, you should use the Firefox browser and select Chinese (Taiwan) (zh-TW) voice for Chinese text-to-speech.
